# Dynamic DLL operations

Bare in mind the following Excel4 macros:

```
'Dynamically call user32 MessageBoxA:
Excel.ExecuteExcel4Macro "CALL(""user32"",""MessageBoxA"",""JJCCJ"",0,""Hello World"",""HELLO WORLD MACRO!"",0)"
```
